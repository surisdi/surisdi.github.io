# Dídac Surís - Personal Website

Personal academic website for Dídac Surís, Research Scientist at Meta.

## Structure

- `index.html` - Main website (single-page design)
- `assets/images/` - Image assets
- `CNAME` - Custom domain configuration

## Local Development

To view the website locally:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Deployment

This website is automatically deployed via GitHub Pages at https://www.didacsuris.com
